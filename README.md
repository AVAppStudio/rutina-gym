# 💪 Rutina Gym - Web App

App web para seguir la rutina de entrenamiento de 4 días. Funciona en móvil (iPhone/Android) desde el navegador.

## Cómo subir a Vercel (paso a paso)

### 1. Subir a GitHub
1. Ve a [github.com](https://github.com) e inicia sesión
2. Pulsa el **+** arriba a la derecha → **New repository**
3. Nombre: `rutina-gym` → **Create repository**
4. Sube todos estos archivos al repo (arrastra y suelta o usa GitHub Desktop)

### 2. Deploy en Vercel
1. Ve a [vercel.com](https://vercel.com) y haz clic en **Sign up with GitHub**
2. Una vez dentro: **Add New → Project**
3. Selecciona el repo `rutina-gym`
4. Deja todo por defecto y pulsa **Deploy**
5. En ~1 minuto tendrás un enlace tipo: `rutina-gym.vercel.app`

### 3. Añadir al iPhone como app
1. Abre el enlace en **Safari**
2. Pulsa el botón compartir (cuadrado con flecha)
3. → **"Añadir a pantalla de inicio"**
4. ¡Aparece como una app más en el iPhone!

## Desarrollo local
```bash
npm install
npm run dev
```
